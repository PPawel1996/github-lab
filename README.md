## Uruchomienie
W katalogu repo:
php -S localhost:8000 -t .

Aplikacja:
http://localhost:8000/frontend/index.html

API:
http://localhost:8000/backend/api/items.php

## Funkcje
- GET lista elementów
- POST dodanie elementu (title min 3 znaki)
- Dane w backend/storage/data.json
